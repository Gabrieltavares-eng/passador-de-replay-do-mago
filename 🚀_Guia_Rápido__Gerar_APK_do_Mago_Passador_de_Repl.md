# 🚀 Guia Rápido: Gerar APK do Mago Passador de Replay

## ⚡ Forma Mais Rápida (5 minutos)

### Passo 1: Criar Conta Expo (Gratuita)
1. Abra https://expo.dev no seu navegador
2. Clique em **"Sign Up"**
3. Preencha email e senha
4. Confirme seu email

### Passo 2: Fazer Upload do Projeto
1. Acesse https://github.com e crie uma conta (se não tiver)
2. Crie um novo repositório chamado `mago-passador`
3. Faça upload dos arquivos do ZIP `mago-passador-final.zip` para o repositório
   - Descompacte o ZIP
   - Copie todos os arquivos para o repositório
   - Faça commit e push

### Passo 3: Conectar GitHub ao Expo
1. Volte para https://expo.dev
2. Faça login com sua conta
3. Clique em **"New Project"**
4. Selecione **"Import from GitHub"**
5. Escolha o repositório `mago-passador`
6. Clique em **"Import"**

### Passo 4: Gerar APK
1. No painel do Expo, clique em **"Build"**
2. Selecione **"Android"**
3. Escolha **"Preview"** como tipo de build
4. Clique em **"Start Build"**
5. Aguarde 5-10 minutos
6. Você receberá um link para baixar o APK

### Passo 5: Instalar no Celular
1. Baixe o APK do link fornecido
2. Abra o gerenciador de arquivos do celular
3. Toque no APK para instalar
4. Se pedir permissão, permita instalar de "fontes desconhecidas"
5. Pronto! O app está instalado
